# Dcon_Protocol_UI
本系統的控制器主要是採用 Dcon 通信協定，將控制器傳送的資料，透過UI介面即時更新顯示，並且將 Miniature Inertial Sensors 接收到的資料，透過UI介面的Remotely operated underwater vehicle的3D圖，達到即時更新轉動的效果。
